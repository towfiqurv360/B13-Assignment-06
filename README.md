# 🎯 DigiTools - Premium Digital Tools Marketplace

## 📖 Description
DigiTools is a modern, responsive, and interactive React web application designed as a marketplace for premium digital products, software, and templates. Users can explore various digital tools, view their features, and add them to an interactive shopping cart. The platform ensures a seamless user experience with dynamic state management and instant visual feedback.

## ⚙️ Technologies Used
- **Frontend Framework:** React.js
- **Styling:** Tailwind CSS, DaisyUI
- **Language:** JavaScript (ES6+)
- **Notifications:** React-Toastify
- **Data Source:** Local JSON

## ✨ Top 3 Key Features
1. **Dynamic Cart Management System:** Users can easily switch between the Product and Cart views. The system automatically calculates total items and total prices in real-time while preventing duplicate item entries.
2. **Interactive Alert Notifications:** Integrated with `react-toastify` to provide beautiful, real-time feedback when users add items to the cart, remove them, or successfully proceed to checkout.
3. **Fully Responsive UI Design:** Built with Tailwind CSS and DaisyUI, ensuring the application looks perfect across all devices (Mobile, Tablet, and Desktop) with a clean, modern aesthetic.
